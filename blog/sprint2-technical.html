Display: inline vs Display: inline-block

Block Elements and Inline Elements

Before we can understand the difference between the display: inline and display: inline-block properties, we must first understand what Block Elements and Inline Elements are. Simply put, Block elements are displayed from top to bottom while Inline elements are displayed in the same line, going from left to right.

Some examples of Block elements are <p> and <div> while examples of Inline elements include <span> and <img>. Check out W3schools for a list of all the Block and Inline elements.

An Example...

One neat trick in web page creation is that we can change the display properties of an element. In a CSS file, we can add a display: command to change a Block element to Inline and vice versa.

In Summary:

Just remember, both display: inline and display: inline-block display things in the same line from left to right. The only difference is display: inline-block preserves the element's original height and width while display: inline does not.
